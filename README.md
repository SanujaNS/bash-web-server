bash-web-server
===============

An HTTP server in Pure Bash.

Watch how it was made on [YouTube](https://www.youtube.com/watch?v=L967hYylZuc).

Usage
-----

```
./bash-web-server [-p port] [-b addr] [-d dir]
```
